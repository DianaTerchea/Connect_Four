# Connect_Four
Se va crea o interfata grafica ce va oferi utilizatorului posibilitatea sa joace po partida de 4 in ROW, atat cu calculatorul, cat si cu un alt oponent. Calculatorul va avea 3 nivele(slab, mediu si avansat). La initializarea se va seta si dimensiunea tablei de joc, respectiv cine incepe primul(calculatorul sau human) daca jucam cu un AI.
